### How to run

- \$ npm i -g run-rs
- to run-rs with its own separate mongo installation
- \$ run-rs --version 4.0.0
- to use the mongo installation you've on your system
- \$ run-rs --mongod
- run-rs usually purges all old data, append --keep to skip purging
- \$ run-rs --mongod --keep
